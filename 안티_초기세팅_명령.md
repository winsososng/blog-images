# 🚀 안티 초기 세팅 명령

> 아래 내용을 안티(Claude Code) 터미널에 붙여넣으세요.
> 안티가 `C:\Claude_Code\blog` 폴더에 진입한 상태에서 실행합니다.

---

## 명령 (복사해서 안티에게 주기)

```
아래 작업을 순서대로 진행해줘.

### 1단계: 폴더 구조 생성
C:\Claude_Code\blog 에 아래 구조를 만들어줘:

blog/
├── input/
│   └── images/
├── output/
│   └── assets/
└── archive/

### 2단계: CLAUDE.md 생성
이 폴더 루트에 CLAUDE.md 파일을 만들어줘.
내용은 내가 별도로 줄게. (첨부 파일 참조)

### 3단계: GitHub repo 생성 및 연결
- repo 이름: blog-images (public)
- 설명: "수학 블로그 이미지 호스팅"
- C:\Claude_Code\blog 을 이 repo에 연결해줘
- .gitignore에 output/*.html 추가

### 4단계: 확인
- 폴더 구조 출력
- git remote -v 출력
- 완료 보고
```

---

## 🔗 그 다음: CLAUDE.md 전달

안티가 폴더 구조를 만들고 나면, CLAUDE.md 파일을 전달해야 합니다.

**방법**: 다운로드 받은 CLAUDE.md 파일을 `C:\Claude_Code\blog\` 에 직접 넣거나,
안티에게 "CLAUDE.md 내용을 만들어줘"라고 하면서 이 파일 내용을 붙여넣으면 됩니다.

---

## 📋 세팅 완료 후 첫 번째 빌드 명령

세팅이 끝나면, 이렇게 명령합니다:

```
input/images/ 에 이미지 15장(page01.png~page15.png)을 넣어놨어.
input/에 master_0편.md (마스터원고)와 work_order.md (작업지시서)도 있어.

CLAUDE.md 읽고, work_order.md 읽고, 아래 순서대로 진행해줘:

1. input/images/ 의 이미지 15장을 GitHub repo의 images/0편/ 폴더에 push
2. push된 이미지의 raw URL을 확보
3. master_0편.md 내용 + work_order.md 지시사항으로 티스토리용 HTML 생성
   - CLAUDE.md의 디자인 시스템, 애니메이션, 칠판 판서 시스템 적용
   - 이미지는 GitHub raw URL로 삽입 (<img src="https://raw.githubusercontent.com/...">)
4. output/post_0편.html 에 저장
5. 완료 보고
```
