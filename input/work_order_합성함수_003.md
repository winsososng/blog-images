#### 🔑 Section 4의 핵심 표 (table-interactive)

**[표3: 꿰매기 vs 이어붙이기 — 메커니즘 차이]** — 반드시 인터랙티브 표로 변환

| 구분 | 꿰매기 (초록) | 이어붙이기 (보라) |
|:---|:---|:---|
| 발생 원인 | 함수 자체의 꺾임 (절댓값 등) | 구간 분리점에서 다른 식이 만남 |
| 조건 | 두 함수의 편차 m + n = 0 | 좌·우 식의 값·미분계수 일치 |
| 본 문제 적용 | x = 2 (\|2x²−8\|의 꺾임) | x = 0 (구간 정의의 분리점) |
| 느낌 | 두 상처가 서로 꿰매줌 | 다른 천 두 장을 매끄럽게 잇기 |

→ 꿰매기 컬럼은 초록 헤더, 이어붙이기 컬럼은 보라 헤더로 시각 구분

### Section 6: 정리 — 꿰매기와 이어붙이기, 다음 편 떡밥
