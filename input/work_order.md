### 섹션별 컴포넌트 매핑

> 마스터원고의 각 섹션에 어떤 다이나믹 컴포넌트를 적용할지 지시.

| 섹션 | 제목 | 컴포넌트 | 비고 |
|:---:|:---|:---|:---|
| 1 | [도입] | `reveal` (기본 등장) | - |

### 사용할 다이나믹 컴포넌트 (체크)

- [ ] `reveal` — 스크롤 등장 (기본, 거의 항상 사용)
- [ ] `math-highlight` — 수식 하이라이트
- [ ] `math-hero` — 핵심 공식 히어로 블록
- [ ] `math-steps` — 수식 단계별 전개
- [ ] `table-interactive` — 인터랙티브 표 (hover)
- [ ] `pattern-discovery` — 패턴 발견 하이라이트
- [ ] `key-message` — 핵심 메시지 박스
- [ ] `accordion` — 펼치기/접기
- [ ] `tabs` — 탭 전환
- [ ] `trap-box` — 함정 경고 (고3)
- [ ] `examiner-block` — 출제자 관점 (고3)
- [ ] `chalkboard` — 칠판 판서 애니메이션 ⭐ (1~3개 권장)
