## 2. 시리즈 컨셉 (전 편 공통)

이 시리즈에는 **두 가지 어둠의 스킬**이 흐름의 황금실로 깔려 있어:
1. **N축 → 꺾인 점 + 개미의 눈** (1편에서 등장하고 확립됨)
2. **라이프니츠식 미분** (2편부터 본격 등장)

1편 톤은 **첫 편답게 친근하고 도입적**으로. 시리즈 전체의 컨셉을 학생에게 친근하게 소개하는 느낌.

## 3. 핵심 아이디어 3개

1. **합성함수는 "두 단계 변환기"** — $x \to g(x) \to f(g(x))$. $x$가 직접 $f$를 흔드는 게 아니라, $g$를 먼저 흔들고 그 흔들림이 $f$로 전달.

2. **꺾인 점만 찾으면 충분하다** — 그래프의 모양은 변화 패턴이 바뀌는 순간들로 결정됨. 0.01씩 다 대입하는 노가다 대신, 꺾인점 + 시작·끝점만 잡고 연결.

3. **개미의 눈** — $g$가 감소 구간이면 $f$ 위의 개미는 거꾸로 걷는 거야. 이 상대적 시점 전환이 합성함수의 본질.

## 7. table-interactive 컴포넌트 사용 ⭐

5번 섹션의 풀이 표는 **table-interactive 컴포넌트**로 구현해줘. 행을 클릭하면 해당 단계 설명이 펼쳐지는 형태.

**표 구조**:

| 행 | 단계 | 내용 |
|:---|:---|:---|
| 1 | 1단계: $x$의 변화율 | 0 → 2 단순 증가, 변곡점 없음. 패스. |
| 2 | 2단계: $g(x)$의 변화율 | $x = 1$에서 꺾임 (V자 꼭짓점). $(1, 0)$ 확정. |
| 3 | 3단계: $f(g(x))$의 변화율 ⭐ | **개미의 눈 등장.** 구간 ① 패스, 구간 ② 정상 통과 → $(\frac{3}{2}, 2)$ 발견. |
| 4 | 4단계: 시작·끝점 | $(0, 2)$, $(2, 0)$ |
| 5 | 5단계: 4점 연결 | $M$자 완성 |

3단계 행에 ⭐ 강조 (시리즈 핵심 통찰).

## 8. trap-box (함정 강조) — 5번 섹션 안

"숨은 꺾인점 발견" 부분을 **trap-box**로 감싸줘. 학생이 가장 놓치기 쉬운 함정이라는 걸 시각적으로 강조.

```
🚨 함정 주의 🚨
시작점·끝점만 보면 구간 ②의 f(g(x))는 0 → 0이야.
그래서 "그냥 0에 머물러 있구나"라고 생각하기 쉬워.
하지만 중간에 g(x) = 1이 되는 순간, f의 정상을 통과해!
숨은 꺾인점 (3/2, 2)를 놓치면 안 돼.
```

## 9. examiner-block (분석 비교) — 6번 섹션

N축 vs 꺾인 점 + 개미의 눈 비교를 examiner-block으로:

| 도구 | 장점 | 단점 |
|:---|:---|:---|
| **N축** | 직관적, 시각화된 풀이 | 기계적, 잔상이 남음 |
| **꺾인 점 + 개미의 눈** ⭐ | 도구 불필요, 정의에 충실, 다음 편들과 연결 | 처음엔 머릿속 시각화 연습 필요 |
