## 1. 들어가며 — 1편의 떡밥, 드디어 회수한다

오늘은 합성함수와 친해지기 시리즈 **3편**이야. 자, 1편에서 아빠가 뭐라고 했지? "꺾인 점은 미분 불가야. 좌미분계수와 우미분계수가 다르니까." 2편에서는 또 뭐라고 했지? "체인룰은 변화의 릴레이야. f(x)가 휘청이면 q(x)도 따라 휘청여."

근데 아빠가 오늘 좀 이상한 이야기를 하나 할게. **"f(x)도 꺾이고 g(x)도 꺾이는데, 둘을 합쳐 놓은 q(x)는 매끄럽다"** — 이게 가능하다는 거야. 헛!!!! 어떻게? 그게 오늘의 주제야. 이름하여 **"꿰매기"**.

> 두 함수의 꺾임이 정확히 반대 방향으로 상쇄될 때,
> 마치 바느질하듯이 두 함수가 서로의 상처를 꿰매준다.

오늘은 이 꿰매기의 정체를 파고들어보고, 마지막엔 2026학년도 3월 학평 22번을 함께 풀어볼 거야. 거기서 "꿰매기"의 쌍둥이 형제 — **"이어붙이기"** 도 만나게 될 거야. 둘은 비슷해 보이는데 메커니즘이 완전히 달라. 시작하자~

## 2. 꿰매기란? — 좌·우 미분계수의 차이가 사라지는 순간

자, 가장 단순한 예부터. f(x)와 g(x) 둘 다 x = a에서 꺾이는 함수라고 해보자. 1편에서 우리는 이런 함수들이 "미분 불가"라고 결론 내렸지. 좌미분계수와 우미분계수가 다르니까.

근데 잠깐. 이걸 더해보면 어떻게 될까? 즉 **q(x) = f(x) + g(x)** 의 좌·우 미분계수를 따져보자.

- **q의 좌미분계수**: f의 좌미분계수 + g의 좌미분계수
- **q의 우미분계수**: f의 우미분계수 + g의 우미분계수

자, 이제 핵심. f의 좌·우 미분계수 차이가 +5이고, g의 좌·우 미분계수 차이가 −5라면? q의 좌·우 미분계수 차이는 +5 + (−5) = **0**. 즉 q는 미분가능해진다는 뜻이야.

다시 말하면, **f도 꺾이고 g도 꺾이는데, 두 꺾임이 정확히 반대 방향으로 같은 크기라서, 합쳐 놓으니까 꺾임이 사라져버린다**는 거지. 우와~ 이게 꿰매기야.

> **꿰매기**: 두 함수가 같은 점에서 똑같은 크기로, 서로 반대 방향으로 꺾여 있을 때, 합쳐 놓으면 꺾임이 상쇄되어 매끄러워지는 현상.

마치 두 함수가 서로 손을 내밀어 상처를 꿰매주는 것처럼 말이야. 이름이 "꿰매기"인 이유야.

## 3. 핵심 공식 — "편차 m + n = 0"

자, 이제 이 직관을 수식 한 줄로 박제해보자. 라이프니츠 정통(2편에서 한참 우려먹었지)을 다시 꺼내자.

g(x)가 x = a에서 미분불가능하다는 걸 가장 깔끔하게 표현하는 방법은:

$$g'(a^-) = k, \quad g'(a^+) = k + m \quad (m \neq 0)$$

쉽게 말해, **"좌극한 미분계수 k에 편차 m이 보태져서 우극한 미분계수가 된다"** 는 거야. 만약 m = 0이라면? 좌·우 미분계수가 같으니까 미분 가능. m이 0이 아니면 미분불가능. 깔끔하지?

마찬가지로 f(x)도 같은 점에서 꺾인다면:

$$f'(a^-) = h, \quad f'(a^+) = h + n \quad (n \neq 0)$$

자, 이제 q(x) = f(x) + g(x)의 좌·우 미분계수를 구해보자.

$$q'(a^-) = h + k$$
$$q'(a^+) = (h+n) + (k+m) = (h+k) + (n+m)$$

좌·우가 같으려면? **n + m = 0**! 이게 끝이야.

[표1: 꿰매기 조건 — 한 줄 요약]

| 상황 | 수식 표현 | 의미 |
|:---|:---|:---|
| g(x)의 꺾임 | g'(a⁺) − g'(a⁻) = m | g의 편차 m |
| f(x)의 꺾임 | f'(a⁺) − f'(a⁻) = n | f의 편차 n |
| q = f + g가 매끄럽기 | **m + n = 0** | 편차가 반대 방향으로 같다 |

말로 풀면 이거야: **"q(x)가 x = a에서 미분가능하려면, g의 좌·우 편차만큼 f의 좌·우 편차가 정확히 반대 방향으로 생기면 된다."** 라이프니츠도 이걸 보면 무릎을 탁 칠 거야. 편차의 합 = 0. 한 줄로 미분가능성을 환원해버렸어.

## 4. 실전 — 2026학년도 3월 학평 22번

자, 이제 진짜 문제로 가보자. 2025년 3월 26일에 시행된 2026학년도 3월 학력평가 공통 22번. 정답률 5.7%짜리 킬러문제야.

> **[2026학년도 3월 학평 22번]**
> 삼차함수 f(x)에 대하여 함수 g(x)를
> 
> $$g(x) = \begin{cases} -f(x) & (x < 0) \\ |f(x)| - |2x^2 - 8| & (x \geq 0) \end{cases}$$
> 
> 이라 하자. 함수 g(x)가 실수 전체의 집합에서 미분가능할 때, **f(−5)** 의 값을 구하시오. [4점]

자, 이 문제를 보면 **미분불가능이 발생할 가능성이 있는 점이 두 군데**야. 어딘 줄 알아?

[표2: 미분불가능 발생 가능 지점 — 두 가지 메커니즘]

| 지점 | 발생 원인 | 해결 메커니즘 |
|:---|:---|:---|
| **x = 0** | 구간 분리점에서 두 다른 식이 만남 | **이어붙이기** (값과 미분계수 일치) |
| **x = 2** | 우측 식의 \|2x² − 8\|이 자체적으로 꺾임 | **꿰매기** (편차 m + n = 0) |

여기서 핵심을 짚고 가자. 두 지점에서 일어나는 일은 메커니즘이 **완전히 달라**. 이게 오늘 포스트의 두 번째 핵심 아이디어야.

### 4-1. x = 2 — 꿰매기로 해결

먼저 메인 디시 x = 2부터. 우측 식 |f(x)| − |2x² − 8|을 봐. 여기서 |2x² − 8|은 2x² − 8 = 0이 되는 x = ±2에서 꺾여. x = 2 근처를 보면, 2x² − 8이 음수에서 양수로 바뀌니까 절댓값이 풀리면서 꺾임이 발생.

x = 2에서 |2x² − 8|의 좌·우 미분계수를 계산해보자. 미분하면 4x인데, 절댓값이 풀리는 방향에 따라 부호가 바뀌어:
- 좌극한: −4·(2) = −8 (왼쪽에서 2x²−8이 음수라 부호 반전)
- 우극한: +4·(2) = +8

따라서 −|2x² − 8|의 편차는 **(−(+8)) − (−(−8)) = −8 − 8 = −16** 이야. 즉 g(x)의 −|2x² − 8| 부분이 x = 2에서 편차 −16의 꺾임을 만들어내고 있어.

이걸 꿰매려면? **|f(x)|가 x = 2에서 정확히 편차 +16의 꺾임을 만들어줘야 해.** 그래야 −16 + (+16) = 0이 되어 매끄러워지니까.

|f(x)|가 x = 2에서 꺾이려면 **f(2) = 0** 이어야 해 (절댓값은 0을 지날 때만 꺾이니까). 그리고 |f(x)|의 좌·우 미분계수 차이가 +16이 되려면, f'(2)의 절댓값의 두 배가 16이어야 하니까 |f'(2)| = 8. 부호까지 맞추면:

$$f(2) = 0, \quad f'(2) = -8$$

이게 꿰매기로 뽑아낸 두 개의 조건식이야!

### 4-2. x = 0 — 이어붙이기로 해결 (꿰매기 ❌)

자, 이제 잠깐. **x = 0에서도 미분불가능 가능성이 있어**. 근데 헛!!! 여기서 일어나는 일은 **꿰매기가 아니야**. 변호사아빠가 원본 포스트에서 "x = 0도 미분불가능 가능"이라고 짚었지만, 이걸 좀 더 정밀하게 다시 보자.

x = 0은 g(x)의 **구간 분리점**이야. 왼쪽은 −f(x)라는 식, 오른쪽은 |f(x)| − |2x² − 8|이라는 다른 식이 만나는 지점. 두 다른 식이 매끄럽게 이어지려면 **(1) 함숫값이 같아야** 하고 **(2) 미분계수가 같아야** 해. 이걸 **"이어붙이기"** 라고 부르자. 꿰매기와 헷갈리지 마.

[표3: 꿰매기 vs 이어붙이기 — 메커니즘 차이]

| 구분 | 꿰매기 | 이어붙이기 |
|:---|:---|:---|
| **발생 원인** | 함수 자체의 꺾임 (절댓값 등) | 구간 분리점에서 다른 식이 만남 |
| **조건** | 두 함수의 편차 m + n = 0 | 좌·우 식의 값·미분계수 일치 |
| **본 문제 적용** | x = 2 (\|2x²−8\|의 꺾임) | x = 0 (구간 정의의 분리점) |
| **느낌** | 두 상처가 서로 꿰매줌 | 다른 천 두 장을 매끄럽게 잇기 |

x = 0에서 이어붙이기 조건을 풀어보자. x = 0 근처에서:
- 좌측 (x < 0): −f(x). 함숫값은 −f(0), 미분계수는 −f'(0).
- 우측 (x ≥ 0): |f(x)| − |2x² − 8| = |f(x)| − 8 (x=0 근처에서 2x²−8 ≈ −8이라 음수). 

함숫값이 같아야 하니까: **−f(0) = |f(0)| − 8**

만약 f(0) > 0이면 |f(0)| = f(0)이라 −f(0) = f(0) − 8 → **f(0) = 4**. (만약 f(0) < 0이면 −f(0) = −f(0) − 8이라 0 = −8이 되어 모순. 그러므로 f(0) = 4 확정.)

이제 미분계수 일치. x = 0 근처에서 f(0) = 4 > 0이니까 |f(x)| = f(x)야. 그래서:
- 좌측 미분계수: −f'(0)
- 우측 미분계수: f'(0) − 0 = f'(0) (|2x²−8|의 미분계수는 x=0에서 0)

같아야 하니까 −f'(0) = f'(0) → **f'(0) = 0**.

자, x = 0에서 이어붙이기로 또 두 개의 조건식이 나왔어!

### 4-3. 조건식 4개로 f(x) 결정

정리하면:

[표4: 4개의 조건식과 출처]

| 조건식 | 출처 | 메커니즘 |
|:---:|:---|:---|
| f(0) = 4 | x = 0 함숫값 일치 | 이어붙이기 |
| f'(0) = 0 | x = 0 미분계수 일치 | 이어붙이기 |
| f(2) = 0 | x = 2에서 \|f\|가 꺾이려면 | 꿰매기 |
| f'(2) = −8 | 편차 +16 만들기 | 꿰매기 |

f(x) = ax³ + bx² + cx + d로 놓고 4개 조건에 대입:
- f(0) = d = 4
- f'(0) = c = 0
- f(2) = 8a + 4b + 4 = 0 → 2a + b = −1
- f'(2) = 12a + 4b = −8 → 3a + b = −2

두 식을 빼면 a = −1, 따라서 b = 1. 결국:

$$f(x) = -x^3 + x^2 + 4$$

마지막으로 f(−5):

$$f(-5) = -(-125) + 25 + 4 = 125 + 25 + 4 = \boxed{154}$$

답은 **154** 야. 정답률 5.7%? 꿰매기와 이어붙이기를 구분할 줄 알면 그렇게 어렵지 않아. 개미의 눈으로 차분히 따라가 보면 길이 보여.

## 5. 곱셈 꿰매기? — 한 줄 미리보기

참고로, 꿰매기는 **덧셈** 합성에만 있는 게 아니야. **곱셈** 합성으로도 꿰맬 수 있어. 그땐 편차가 더하기가 아니라 **편차비율**로 바뀌어. 즉 m + n = 0이 아니라 **m × n = 1** 이어야 해.

근데 신기하게도 이 곱셈 꿰매기로 출제된 기출은 아직 없어. 아빠가 찾아봤는데 나오질 않더라ㅠㅠ. 그 말은? 앞으로 나오면 변별력이 어마어마하다는 이야기. 이건 시리즈 5편 변신로보트 편에서 다시 만나자.

## 6. 정리 — 꿰매기와 이어붙이기, 다음 편 떡밥

오늘의 핵심을 한 번에 정리하면:

> **꿰매기**: 두 함수가 같은 점에서 반대 방향으로 똑같이 꺾이면, 합쳐 놓은 함수는 매끄러워진다 — 편차 m + n = 0
>
> **이어붙이기**: 구간 분리점에서 두 다른 식이 만날 때, 값과 미분계수가 일치해야 매끄럽다 — 메커니즘이 완전히 다름

이 두 개를 구분할 줄만 알면 미분가능성 문제는 더 이상 무섭지 않아.

자, 그리고 마지막으로. 오늘 우리는 두 함수를 **꿰매서 매끄럽게 만드는** 법을 배웠어. 근데 잠깐... 만약 거꾸로라면? **매끄러운 함수를 일부러 쪼개서 풀어내는** 방법도 있지 않을까?

헛!!!! 그게 바로 **다음 편 — 4편의 주제**야. 이름하여 **치환적분**. 꿰매기의 거울이지. 라이프니츠 정통을 거꾸로 돌려서 푸는 거야. 합성함수 미분이 "두 함수를 묶어서 미분"이라면, 치환적분은 "한 함수를 두 함수로 풀어서 적분"이거든. 우와~ 어떤 그림인지 궁금하지?

다음 편에서 만나자~
