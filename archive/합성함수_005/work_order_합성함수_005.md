## ⚠️ Anti 작업 시 주의사항

1. **Python 빌드 스크립트 쓰지 말고 HTML 파일을 직접 작성할 것**
2. **이미지는 반드시 GitHub raw URL로 삽입** (형식: `https://raw.githubusercontent.com/winsososng/blog-images/main/images/합성함수_005/파일명.png`)
3. **수식은 인라인 텍스트로 처리** (LaTeX PNG는 이미 input/ 폴더에 있음, MathJax 굳이 안 써도 OK)
4. **칠판 판서는 섹션 4에 1개만** (과용 금지)
5. **이모지는 최소화** — 변호사아빠 톤은 글 자체의 따뜻함으로 표현
6. **f(x) 표기는 그대로** — 위첨자가 어려우면 `x^2` 형태 OK
7. **반응형 디자인 필수** (모바일 / 태블릿 / 데스크탑 전부 OK)
8. **이미지 크기 조정** — 수식 PNG는 max-width: 600px 권장, 그래프는 max-width: 500px 권장
