  - 좌측: Δ (변량) | 우측: d (무한소)
    | Δ (변량) | → | d (무한소) |
    |:---:|:---:|:---:|
    | Δx | → | dx |
    | Δy | → | dy |
    | Δt | → | dt |
    | **Δf(x)** | → | **df(x)** |
    | **Δg(x)** | → | **dg(x)** |
    | **Δ(x²)** | → | **d(x²)** |
    | **Δ(sin x)** | → | **d(sin x)** |
  - 강조 색: dx, dy, dt는 회색(기본), df, dg, d(x²), d(sin x)는 앰버(주목)

### 섹션 5: 라이프니츠 정통 풀이 — 치환은 사실 필요 없다 ⭐⭐ 메인 디쉬

- **컴포넌트**: `tabs` (경로 A | 경로 B) 또는 좌우 비교 카드
- **5-A: 경로 A — 라이프니츠 정통 (치환 없는 본질 풀이)** ⭐ 메인:
  - **이미지 삽입 (가운데 메인)**:
    - `formula_002_chain_derivation.png` — `math-hero` 처리
    - 캡션: "라이프니츠 정통 풀이의 핵심 3단계. 1을 곱하고, 분수의 곱셈 규칙으로 분모 자리 바꾸기. 치환 없이 합성함수 미분 공식이 그대로 튀어나온다."
  - **각 단계별 설명** (이미지 위 또는 아래로 풀어서):
    1. 원식: $\frac{df(g(x))}{dx}$
    2. 1을 곱한다: $\frac{dg(x)}{dg(x)} = 1$
    3. **분수의 곱은 분자끼리·분모끼리, 곱셈 순서는 무관**이므로 분모 자리 바꾸기
    4. 두 분수의 곱으로 다시 쪼개면 **합성함수 미분 공식 완성**
  - **칠판 판서 (chalkboard) 2번** ⭐:
    - 위치: 도출 끝난 직후
    - 내용:
      ```
      🔑 치환은 본질이 아니다.
      
      g(x)=t는 손가락 편의일 뿐.
      라이프니츠는 치환 없이도 풀린다.
      ```
    - 색상: "치환은 본질이 아니다" 노랑+밑줄, "손가락 편의" 빨강, "치환 없이" 파랑
    - 별표 팝(★) 효과 ON
    
- **5-B: 경로 B — 학교에서 흔히 쓰는 치환식 (손가락 편의)**:
  - 컴포넌트: 작은 카드 또는 `accordion` (펼치기/접기)
  - 라이프니츠 치환식: $\frac{df}{dx} = \frac{df}{dt}\cdot\frac{dt}{dx}$, $t=g(x)$
  - **명확히 표시**: "본질이 아니라 표기 편의"
  - 이미지 삽입 (경로 B 카드 안):
    - `original_005_substitution_example.png`
    - 캡션: "학교에서 자주 쓰는 치환 예제. 본질은 표기 편의일 뿐이다."

- **5-실전: 진짜 문제로 굴려보자**:
  - 컴포넌트: `math-steps` (Step 1~4를 스크롤로 차례차례 드러나게)
  - 문제 박스 (`examiner-block` 또는 핵심 메시지):
    ```
    [문제] f(x) = x³ - 3x², g(x) = x² + x + 1 일 때,
           f(g(x))를 미분하라.
    ```
  - **Step 1**: 구하려는 것은 $\dfrac{df(g(x))}{dx}$
  - **Step 2**: 1 곱하고 분모 자리 바꾸기 → $\dfrac{df(g(x))}{dg(x)} \cdot \dfrac{dg(x)}{dx}$
  - **Step 3**: 두 조각 각각 계산
    - $\dfrac{df(g(x))}{dg(x)} = 3g^2 - 6g = 3g(g-2)$
    - $\dfrac{dg(x)}{dx} = 2x + 1$
  - **Step 4**: 곱하고 $g = x^2+x+1$ 대입
    - 최종 답을 `math-hero` 강조:
      $$
      \frac{df(g(x))}{dx} = 3(x^2+x+1)(x^2+x-1)(2x+1)
      $$
  - 마무리: "외운 공식 하나 없이, **치환 한 번도 없이**, 분필가루 df, dg를 변수처럼 굴려서 답이 나왔지." → 굵게 강조

### 섹션 6: 곱미분과 헷갈리지 마라 — 곱은 더하고, 합성은 곱한다

- **컴포넌트**: `table-interactive` (인터랙티브 비교표)

| 구분 | 식 | 미분 |
|:---:|:---:|:---:|
| **함수의 곱** | $f(x) \cdot g(x)$ | $f'(x)g(x) + f(x)g'(x)$ → **더하기** |
| **합성함수** | $f(g(x))$ | $f'(g(x)) \cdot g'(x)$ → **곱하기** |

- **칠판 판서 (chalkboard) 3번** ⭐ — 시리즈 시그니처 슬로건:
  - 위치: 비교표 직후
  - 내용:
    ```
    🔑 곱은 더하고,
       합성은 곱한다.
    ```
  - 색상: "곱"과 "합성" 단어를 각각 노랑/파랑으로 구분, 동사 "더하고/곱한다"는 빨강(주의)
  - 굵게, 밑줄 효과 ON
- **곱미분 예외 회수 메시지** (섹션 2 떡밥 회수):
  - "2편 인트로에서 아빠가 말했지? '딱 하나 예외가 곱미분이다'라고. 바로 이거야." → `key-message` 박스

## ⚠️ Anti 빌드 시 주의사항

1. **Python 빌드 스크립트 사용 금지** — HTML 파일을 직접 작성할 것
2. **`<!DOCTYPE>`, `<html>`, `<head>`, `<body>` 태그 사용 금지** — 티스토리가 자동 추가
3. **이미지는 반드시 GitHub raw URL 사용** (push 후 raw URL로 `<img src="...">` 삽입)
4. **MathJax 수식**: `$...$` (인라인), `$$...$$` (블록) — 둘 다 사용
5. **마스터원고 수학 내용 임의 변경 금지** — 변호사아빠 시그니처 톤 ("아빠가~") 그대로 유지
6. **원본 4번 이미지(`original_004_chainrule_proof.png`)는 사용하지 않음**
7. **시리즈 정합성**:
   - 1편의 "N축 어둠의 스킬", "개미의 눈" 표현이 자연스럽게 회수되어야 함
   - 분필가루 비유가 시리즈 시그니처임을 디자인적으로도 강조
   - **"치환은 본질이 아니다"가 2편의 핵심 폭로**임을 디자인적으로 부각
