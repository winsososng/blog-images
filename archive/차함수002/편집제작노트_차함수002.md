## 🎯 출제자 관점 분석 (고3용 필수!) ⭐

### Q1. 출제자는 왜 이 문제를 냈을까?
3차함수 문제에서 출제자가 진짜 보고 싶은 것은 "복잡한 식을 단순화할 줄 아느냐"이다. ax³+bx²+cx+d 형태 그대로 미정계수 연립하는 학생 vs 차함수 스킬로 거리곱 형태로 변환하는 학생을 가르려는 것. 결국 "수식적 센스"를 검증하는 문제.

### Q2. 핵심 출제 포인트는?
- 3차함수와 직선이 3점에서 만나는 조건이 주어졌을 때, f(x) = h(x-a)(x-b)(x-c) + g(x)로 변환할 수 있는가
- 거리곱 + 보정으로 특정 점에서의 함수값/미분계수를 빠르게 구할 수 있는가
- 접선 조건이 주어졌을 때, 접점에서의 차함수 표현 (중근 처리)을 할 수 있는가

### Q3. 출제자가 노린 함정은?
| 함정 | 설명 | 회피법 |
|:---|:---|:---|
| 무대포 연립 | ax³+bx²+cx+d에 조건 대입 → 4원 연립 → 시간 폭망 | 조건에 "직선과 만남"이 보이면 즉시 차함수 스킬 |
| 보정 누락 | 차함수로 내려간 뒤 g(x) 되돌리기를 빼먹음 | "내려오기 + 되돌아가기"를 항상 세트로 |
| 최고차계수 무시 | h를 1로 착각하거나 빼먹음 | h(x-a)(x-b)(x-c)에서 h 확인 필수 |
| 접선=중근 미인식 | 접하는 점에서 (x-a)² 처리를 못함 | "접한다 = 중근"임을 즉시 반영 |

### Q4. 같은 포인트로 가능한 변형은?
- "f(x)-g(x)의 극대/극소" 물어보기 (차함수 자체의 성질 활용)
- "두 함수로 둘러싸인 넓이" → 차함수의 적분 = 거리곱 적분
- "접선 조건 + 나머지 교점" → 차함수 스킬 + 중근 처리의 콤보
- "f(a)의 값을 구하시오" 형태 → 거리곱 + 보정의 직접 활용
