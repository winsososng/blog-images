## 0. 핵심 한 줄

3차함수 문제에서 "직선과 만난다"는 조건이 보이면, 미정계수 연립 말고 **차함수 스킬**을 써라. 계산량이 80% 줄어든다.

---

## 1. 차함수 스킬 요약

넌 이미 알겠지만, 확실히 정리하자. 3차함수 $f(x)$와 직선 $g(x)$가 $x = a, b, c$에서 만나면:

$$f(x) = h(x-a)(x-b)(x-c) + g(x)$$

$h$는 최고차계수. 이 식은 항등식이야. 교점 정보가 이미 인수에 들어가 있으니까, 여기서 뭘 구하든 거리곱 한 방이면 끝.

핵심은 이거야. **"내려오기(차함수로 변환) → 관찰(거리곱) → 되돌아가기(보정)"**. 이 프로토콜만 익히면 3차함수 계산의 대부분이 암산 수준으로 내려와.

---

## 2. 거리곱 + 보정 실전 공식

### 함수값

$$f(d) = h(d-a)(d-b)(d-c) + g(d)$$

거리를 곱하고, 직선값을 더하면 끝.

### 미분계수

$x = a$에서의 미분계수:

$$f'(a) = h(a-b)(a-c) + g'(a)$$

$(x-a)$ 항이 미분되면서 1이 되고, 나머지 $(x-a)$ 포함 항은 0. 결국 남은 거리곱에 직선 기울기 보정.

### 접선 조건 (x = a에서 접할 때)

접한다 = 중근이니까:

$$f(x) = h(x-a)^2(x-c) + g(x)$$

여기서 $g(x)$는 $x = a$에서 $f(x)$에 접하는 직선.

## 🎯 출제자의 의도 ⭐

### Q. 출제자는 왜 이 문제를 냈을까?

출제자가 보고 싶은 건 딱 하나야. **"이 학생이 복잡한 식을 단순화할 줄 아는가?"** $ax^3 + bx^2 + cx + d$ 형태 그대로 미정계수 연립하는 학생과, 차함수 스킬로 거리곱 형태로 변환하는 학생을 가르려는 거지. 결국 **수식적 센스**를 검증하는 문제야.

### Q. 핵심 출제 포인트는?

1. **차함수 표현 전환**: "직선과 3점에서 만난다" → $f(x) = h(x-a)(x-b)(x-c) + g(x)$로 즉시 변환할 수 있는가?
2. **거리곱 + 보정 계산**: 특정 점에서 함수값/미분계수를 거리곱으로 빠르게 구할 수 있는가?
3. **접선 = 중근 처리**: 접하는 조건이 주어지면 $(x-a)^2$ 처리를 바로 할 수 있는가?

### Q. 출제자가 노린 함정은?

[표1: 출제자 함정 분석]

| 함정 | 왜 틀리기 쉬운가 | 회피법 |
|:---|:---|:---|
| 무대포 연립 | "직선과 만남" 조건을 보고도 습관적으로 미정계수 4개 연립 | 조건에 "직선과 만남"이 보이면 즉시 차함수 스킬 |
| 보정 누락 | 차함수에서 관찰한 값을 답으로 착각 (g(x) 복원 안 함) | "내려오기 + 되돌아가기"를 항상 세트로 |
| 최고차계수 h 누락 | $h=1$로 착각하거나 빼먹음 | 문제에서 최고차계수 확인 필수 |
| 접선 = 중근 미인식 | 접하는 조건에서 $(x-a)^2$ 처리를 못함 | "접한다 = 중근"은 반사적으로 |

### Q. 이 포인트로 가능한 변형은?

- **변형 1**: "f(x) - g(x)의 극대값/극소값 구하기" → 차함수 자체의 성질 활용
- **변형 2**: "두 함수로 둘러싸인 넓이" → 차함수의 정적분 = 거리곱 형태 적분
- **변형 3**: "접선 + 나머지 교점" → 중근 처리 + 근과 계수 관계
- **변형 4**: "$f(a)$의 값을 구하시오" → 거리곱 + 보정 직접 활용

> 💡 **핵심**: "직선과 만난다" 조건이 보이면 차함수 스킬. 안 쓰면 시간 폭망이야.

## 4. 정리

차함수 스킬 = 직선을 기준으로 3차함수를 거리곱 형태로 변환.

$$f(x) = h(x-a)(x-b)(x-c) + g(x)$$

**함수값**: 거리곱 + 직선값. **미분계수**: 거리곱 미분 + 직선 기울기. 항상 **내려오기 → 관찰 → 되돌아가기**.

출제자는 네가 이걸 쓸 줄 아는지를 본다. 안 쓰면 시간이 부족해져서 결국 틀린다. 이게 상위권을 가르는 스킬이야.
